# What is a race condition?
A race condition occurs when multiple requests access or modify the same data at the same time, and the final result depends on the order in which they happen.

# What is atomic operation?
An atomic operation is an operation that is completed in a single step, and cannot be interrupted by other operations.

# What is a transaction?
A transaction is a sequence of operations that are executed as a single unit of work. If any of the operations fail, the entire transaction is rolled back, and the system returns to its previous state.
